# LinkedIn Memory Badge 🧠

Instantly remember how you know someone on LinkedIn without leaving the tab. 
A premium, private-by-design context card that sits right in your viewport.

## How to Install (Takes 30 seconds)
1. **Download** this repository as a ZIP file and extract it to your computer.
2. Open Google Chrome and type **`chrome://extensions/`** into the address bar.
3. Toggle on **Developer mode** in the upper-right corner.
4. Click the **Load unpacked** button in the upper-left corner.
5. Select the extracted folder containing the extension files.

## How it Works
All your contextual data is saved locally on your machine using `chrome.storage.local`. 
No servers, no tracking, completely private.
